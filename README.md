This repository now contains a minimal React + Vite scaffold to start building the Figma OCI design system UI.

Getting started (macOS / `zsh`):

1. Install dependencies:

```bash
cd /Users/adishanbhag/Desktop/MCP-OCI-Design-System/Figma-OCI-design-system
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
npm run preview
```

Files added:
- `package.json`: scripts and deps
- `vite.config.js`: Vite + React plugin
- `index.html`: app root
- `src/`: `main.jsx`, `App.jsx`, `index.css`
- `public/favicon.svg`
- `.gitignore`
